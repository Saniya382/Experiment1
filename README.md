# Experiment1
"Python program demonstrating all major list operations including creation, access, modification, and built-in methods.”
This repository contains Experiment 1 of the Machine Learning Lab, which focuses on understanding and demonstrating all major operations on Python lists.

The program covers fundamental list concepts such as:
- Creating lists
- Accessing elements (indexing and slicing)
- Updating and deleting elements
- Using built-in methods like append(), extend(), insert(), remove(), pop(), sort(), reverse(), and count()
- Looping through lists
- List concatenation and basic operations

This experiment helps build a strong foundation in Python programming, which is essential for Machine Learning and data handling tasks.
